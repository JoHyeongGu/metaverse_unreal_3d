=== <Blueprint> ===
1. Gate
    1) Enter: 문을 들어간다.
    2) Open: 문을 연다.
    3) Close
    4) Toggle: 문이 열려 있으면 닫고 닫혀 있으면 연다.
    5) Exit: 다음 흐름
    6) Start Closed (bool): 닫힌 상태로 시작할지, 열린 상태로 시작할지 결정

2. Multi Gate
    1) Out 0 ~ n: 출구가 여러 개가 될 수 있다.
    2) Is Random (bool): 순차대로 출구를 나가지 않고 랜덤한 순서로 진행된다
    3) Loop (bool): 반복해서 진행된다
    4) Reset (int): 게이트를 초기화한다.

3. Do Once
    1) Start Closed (bool): 닫힌 상태로 시작할지, 열린 상태로 시작할지 결정
    2) Reset: 노드 초기화

4. Do N
    1) N: 반복 횟수
    2) Counter: 현재 인덱스

<Enum 타입>
(Enum 파일 생성 -> 블루프린트 / 열거형)

Enum 타입을 출력할 때는, Print String으로 출력 가능 (Print Text로는 불가능)
